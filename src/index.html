Como funciona:

Esta é uma calculadora para saber quanto vale sua hora no freelancer. No formulário ao lado você deverá preencher (em números) os valores que cada campo solicita.
Para mais detalhes, utilize as setas abaixo para checar o que cada campo significa.

Horas Diárias:

Nada mais é que as horas diárias que você irá se envolver com o projeto. Vale lembrar que em empresas, as horas diárias "normais" são 8hrs por dia, geralmente, de sete horas da manhã às cinco horas da tarde.
Mas aqui você decide junto ao seu cliente.

Dias Efetivos:

Dias efetivos serão os dias que você estará presente no desenvolvimento do projeto. Isso novamente vai depender da sua conversa com o cliente, vai trabalhar 7 dias? 15 dias?
Após isso ser conversado e bem alinhado, deverá preencher este campo.

Dias de Férias:

Quantos dias de férias decidiu tirar? Vai curtir uma praia para sair dessa rotina de tecnologias? Este campo é justamente para esses dias, talvez não pegará férias, somente feriados, mas também atende.

Valor do Projeto:

Após toda a conversa entre você e o cliente, ficou claro quanto tempo você deverá utilizar-se de códigos.
Como tempo é dinheiro, você também provavelmente sabe quanto o projeto irá custar, então este será o valor inserido aqui.
OBS: utilize somente números, a calculadora não entende letras ou símbolos.
